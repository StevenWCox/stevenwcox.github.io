Rest.HttpMockResponse

global class HttpMockResponse implements HttpCalloutMock
Descriptionan HttpCalloutMock implementation for testing web service callouts

Can handle multiple endpoints, multiple methods per instance, and multiple responses per endpoint/method

Usage:

   // test simple callout (may be used multiple times)
   lws.Rest.mock(new lws.Rest.HttpMockResponse('b1', Rest.HTTP_SUCCESS));
   ... // run Apex that performs callouts and validate results
 

   // test several different callouts (each may be used only once)
   HttpMockResponse m = new lws.Rest.HttpMockResponse('GET', 'https://google.com', 'b1', lws.Rest.HTTP_SUCCESS);
   m.add('GET', 'https://google.com', 'b2', lws.Rest.HTTP_SUCCESS); // test second GET callout
   m.add('PUT', 'https://google.com', 'b3', lws.Rest.HTTP_SUCCESS); // test PUT to Google
   m.add('GET', 'https://amazon.com', 'b4', lws.Rest.HTTP_SUCCESS); // test GET callout to Amazon
   lws.Rest.mock(m);
   ... // run Apex that performs callouts and validate results
