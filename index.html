ApexBoost Library

Welcome to the ApexBoost Library - a collection of time-saving, high-quality code for use in any Salesforce project. Below is a summary of the major features. Additions will be released often!

Error Logging

The lws.Logger generates log records when errors occur in custom code. Errors will be saved even if a subsequent exception is thrown - guaranteeing that errors will be noticed. Stakeholders can schedule reports or create other workflows to receive notifications in real time.

LWC Build Standards

The ApexBoost Library is built using our high internal standards and Salesforce best practices. With each release, we guarantee:

  • A full suite of succeeding unit tests with 100% coverage for easy deployment
  • Clean PMD and CPD reports, guaranting security, scalablity, and maintainability.
  • A clean security scan.
  • Internal diagnostics and error reporting using the library itself.

Version history

1.0 (Initial library release)

  • Logging to a custom object: includes log level configuration and batch process logging.
  • A log cleaner batch process that can be scheduled to prevent buildup of log records.
