AssertException

global inherited sharing class AssertException extends Exception
Description This exception is thrown by the lws.Assert methods. Unlike System.AssertException, lws.AssertException can be caught by custom code. This makes it possible for you to use lws.Assert within your code and write unit tests that validate those assertions.
