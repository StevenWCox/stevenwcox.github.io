AssertException

global inherited sharing class AssertException extends Exception
DescriptionThis exception is thrown by the lws.Assert methods. Unlike System.AssertException, lws.AssertException
  • Can be caught by custom code. This makes it possible for you to use lws.Assert within your code, and write unit tests that validate those assertions.
  • Can be disabled with lws.Settings, allowing you to utilize assertions during development and debugging, but not incur the overhead in production.
